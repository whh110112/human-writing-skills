# Cool18 热门小说研究

## 范围与边界

本研究于 2026-07-17 查看禁忌书屋的公开[人气热贴](https://www.cool18.com/bbs4/index.php?app=forum&act=hot)、当前帖子页面和[2025 年热门回顾](https://www.cool18.com/bbs4/index.php?act=threadview&app=forum&tid=14527517)。只记录标题、排名、页面结构、章节统计和短范围技法判断，不保存或复制小说正文。

榜单按阅读量反映注意力，不等于文笔排名。站点提供 AI 标记和过滤功能，但零票、未标记或未进入过滤名单都不能证明作者身份。本项目因此不做“是否 AI 创作”的断言，而采用两层筛选：站内公开标记作为弱信号；确定性规则和人工结构复核只判断文本是否适合作为正向技法样本。

## 从正向样本保留什么

- **读者承诺明确**：标题、题材、人物关系和开篇很快说明本篇要提供的核心冲突。
- **熟悉场景承载压力**：家庭、职场、村镇、门派等场景先提供可理解的身份、资源或关系摩擦，再扩展设定。
- **对白具有交易性**：有效对白会改变许可、信息、债务、风险、地位或关系，而不是只确认气氛。
- **章节留下可继承残留**：上一场产生的物件、秘密、损失、承诺或关系温度进入下一场。
- **情绪悬念强于说明悬念**：读者继续阅读往往不是等待百科答案，而是等待人物会接受、拒绝、隐瞒、失去或反击什么。
- **毛边可以存在**：段落长短和措辞不必统一精修，但指代、位置、关系、行动结果和场景压力必须清楚。

站内年度回顾把受欢迎作品概括为文笔流畅、剧情起伏和持续更新三项中至少具备两项。这个观察只作为读者偏好的站内总结，不降低本项目对逻辑、物理和关系连续性的要求。

## 从负向样本排除什么

部分高点击文本同时集中出现以下模式，因此只作为负向对照，不提炼为“热门文风”：

- 章节开头一次交代精确时刻、完整地标、天气或彩色光线、全套穿着、气味和概括情绪；
- 连续章节都从新的风景镜头重启，上一章造成的压力没有进入开场；
- 反复使用“不知道为什么”“莫名”“不是这种而是那种”“像是又像是”解释人物感受；
- 比喻、形容词和身体感受很密，但人物的选择、许可、风险和关系没有改变；
- 章尾突然放入无来源的新威胁，只为了制造钩子。

高点击与这些问题可以同时存在。项目学习的是可迁移的叙事机制，不是复制排行榜作品的表面词汇。

## 落地模块

| 发现 | 实现 |
| --- | --- |
| 多章只铺气氛、不兑现承诺 | `chapter-momentum-audit`、按需 `momentum` profile |
| 电影式开场套餐 | `scene-entry-audit`、`OPEN002` |
| 模糊内心解释反复出现 | `EMO003` |
| 每章从天气和穿着重启 | `RESET001` |
| 多章稿件需要独立审查但没有前章账本 | `momentum` 与要求 `--context` 的 `serial` 分离 |

`momentum` 不进入默认 `full`、`--review` 或 `--deep-review`。`pipeline --auto` 只有检测到两个以上章节标题或重复续篇标记时才加载；电影式开场问题仍交给独立的 `texture` profile。
