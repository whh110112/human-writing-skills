# Imagery Load Audit

## Aim

Prevent prose from sounding manufactured because every object, sound, sensation, and
emotion receives a metaphor or multiple sensory restatements.

## Activation Gate

Use when imagery, sensory description, emotional embodiment, or ornamental prose is
dense enough to compete with action. Do not load for plain technical, legal, news, or
academic text unless explicitly requested.

## Image Budget

- Give a beat one dominant image or sensory channel when possible.
- Keep a comparison only if it changes perception, choice, tone, or later meaning.
- Prefer an image system that belongs to the viewpoint character's experience.
- Do not personify every object or make every sound perform the same emotion.
- Repeated motifs must change meaning, not merely repeat wording.

## Show-Then-Gloss Test

Flag a visible action, body cue, or sensory detail immediately followed by a sentence
that explains the same emotion. If removing the gloss preserves meaning, cut it. Keep
the explanation only when the character misreads, revises, or complicates the evidence.

## Output

```text
Imagery Load Review
- Dense image or sensory cluster:
- Competing or generic comparisons:
- Show-then-gloss duplication:
- Motif with changed meaning:
- Image to keep and material to cut:
```

The target is not sparse prose. It is selective attention.
