# Appearance and Prop Continuity Skill

## Aim

Prevent clothing, shoes, accessories, injuries, carried objects, and daily details from changing without a scene event.

## Required State

Track appearance and props with a state table:

| Character | Clothing | Shoes | Hair/makeup | Injuries/body state | Carried items | Last confirmed |
| --- | --- | --- | --- | --- | --- | --- |
|  |  |  |  |  |  |  |

## Change Gate

An item or appearance detail may change only if one of these is written:

- the character changes clothes
- time passes and a new outfit is established
- the item is lost, handed over, hidden, damaged, or put down
- weather or action plausibly alters the detail
- a narrator corrects a prior mistaken observation

If no change gate appears, the previous detail remains true.

## Daily Detail Checks

Track details that often drift:

- short skirt vs long skirt
- flats vs heels
- coat on vs coat removed
- wet hair vs dry hair
- bag on shoulder vs bag left behind
- phone in hand vs phone in pocket
- blood, dust, rain, makeup, bandage, glasses

## Avoid

- using clothing as mood decoration if it contradicts earlier state
- changing footwear to fit a later action
- forgetting items that limit movement
- letting injuries disappear because the scene moved on

## Revision Check

For every appearance or prop detail in the new passage, ask: was this already established, newly changed on-page, or an error?
