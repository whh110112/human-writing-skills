# Logic and Causality Audit

## Aim

Find plot and argument failures that remain even when individual sentences read well.
Audit evidence, cause, time, knowledge, motive, rules, resources, and consequences
before suggesting prose changes.

## Evidence Map

For every important event or claim, extract:

```text
trigger -> actor/claim -> action/reasoning -> immediate result -> later consequence
```

Also record:

- what must already be true
- who knows the required fact
- which rule, resource, ability, or constraint permits the step
- whether enough time and access exist
- which later passage depends on the result

## Contradiction Types

- Missing cause: an event happens because the plot needs it, with no trigger or motive.
- Reversed cause: the text uses a later fact to justify an earlier decision.
- Knowledge leak: a character acts on information they have not learned.
- Timeline conflict: travel, recovery, communication, or preparation takes impossible time.
- Rule exception: a power, institution, contract, technology, or social rule changes for convenience.
- Resource creation: money, evidence, equipment, access, authority, or opportunity appears without source.
- Consequence dropout: a cost, injury, promise, crime, discovery, or public event stops mattering.
- False inference: the evidence supports suspicion, but the draft treats it as certainty.
- Goal mismatch: an action undermines the actor's established goal without a new motive.

## Audit Method

1. Build a chronological event table.
2. Mark each event's cause, required knowledge, resource, rule, and consequence.
3. Separate confirmed contradiction from missing explanation and intentional mystery.
4. Check whether a later paragraph quietly repairs or changes the fact.
5. Recommend the smallest bridge, setup, reordered event, limited inference, or consequence.

Do not repair every mystery. A withheld explanation is valid when the event remains
possible and the text signals uncertainty.

