# Paragraph Rhythm Audit

## Aim

Detect manufactured pace created by equal paragraph blocks or long runs of isolated
one-line fragments. Paragraph breaks should mark changes in speaker, attention,
pressure, time, or consequence.

## Activation Gate

Use when the draft contains many short paragraphs, fragment chains, or a request about
chapter cadence. Do not treat every short paragraph as an error.

## Audit

- Map each paragraph to its function: orient, act, speak, react, reveal, reflect,
  compress, or turn.
- Flag runs where several short paragraphs perform the same function.
- Check whether isolated names, locations, times, or emotion labels create false impact.
- Check whether long blocks hide multiple turns that need separation.
- Compare paragraph size with pressure: shock can contract; bargaining, observation,
  and memory often need room.

## Repair

Merge fragments that share one action, split only at a real turn, and preserve an
isolated line only when surrounding rhythm makes it land.
