# Controlled Drift Skill

## Aim

Break the model's over-smooth reasoning by allowing brief, purposeful detours that feel like a human mind moving through memory, association, doubt, or irritation.

## Use When

- fiction, memoir, essay, and reflective self-media writing feel too linear
- the transition between two ideas is technically correct but emotionally flat
- a character or narrator needs a private current beneath the main action

## Techniques

- Insert a short associative turn: an object, smell, weather detail, old phrase, or remembered scene that echoes the main point without explaining itself.
- Let a sentence correct itself when the narrator's first wording is too clean.
- Preserve a small open question instead of resolving every thought.
- Use a fragile inference: show the evidence fragment before the conclusion.
- Return to the main line quickly. The detour should bend the passage, not replace it.

## Avoid

- random surrealism without emotional or thematic pressure
- fake confusion that hides missing logic
- detours in hard news, legal, or strict technical prose
- overusing hesitation in every paragraph

## Revision Check

The detour should add subtext, memory, pressure, or voice. If removing it changes nothing, cut it.
