# Argumentative Essay Skill

## Aim

Build a persuasive argument that sounds like a thoughtful person, not a template.

## Structure

- Open with a concrete tension, not a universal claim.
- State the thesis plainly.
- Develop one claim per section.
- Use evidence, example, or reasoning before conclusion.
- Include a serious counterargument and answer it fairly.

## Style Controls

- Prefer precise verbs over intensifiers.
- Use transitions that name the logical relationship.
- Let uncertainty appear where the evidence is limited.
- End by sharpening the implication, not repeating the thesis.

## Avoid

- "In today's society" openings
- three-point essay scaffolding that announces itself
- inflated certainty
- strawman counterarguments
