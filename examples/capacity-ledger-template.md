# Capacity Ledger Template

Use this template when the scene depends on who or what can fit on a physical resource.

## Physical Resources

| Resource | Mode | Capacity rule | Occupants/items | Access path | Transformation history |
| --- | --- | --- | --- | --- | --- |
| rear-left zone | unknown | unknown until text establishes seat/bench/bed/floor |  |  |  |
| stool | standing surface | one adult briefly if stable, otherwise unsafe |  | climb/step down |  |
| bed | lying surface | depends on size and posture |  | side access |  |
| motorcycle saddle | seated support | driver plus possible passenger if established |  | mount/dismount |  |

## Capacity Rules

- Unknown capacity should stay unknown until the prose establishes it.
- Sharing is allowed only when the resource mode supports sharing or the prose describes crowding, layering, lap-sitting, lying down, or transformation.
- A transformation must be written before the resource is used in its new mode.
