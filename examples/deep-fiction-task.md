# Deep Fiction Task Example

Use this with:

```powershell
python -m humanwriting.cli build `
  --style fiction `
  --module controlled-drift `
  --module embodied-emotion `
  --module vocal-rhythm `
  --module cultural-anchors `
  --review `
  --context examples/story-ledger.md `
  --task "Write the next 900 words. Begin outside the shuttered customs hall at dawn. Lin Qiao must use one fragile inference instead of certainty, and the scene should end with one unresolved pressure."
```

## Editorial Intention

The passage should not sound perfectly engineered. It should feel like a close third-person narrator following a tired, frightened person who is still trying to appear competent.

## Required Texture

- one short associative detour tied to the river quarantine
- one bodily sign of fear that Lin Qiao tries to hide
- one cultural anchor from Yanxi's port bureaucracy
- one sentence that corrects itself without becoming melodramatic
- one final image or line that keeps the conflict alive

## Do Not

- explain Captain Wei's full motive yet
- solve the ferry clue
- invent a new power
- summarize Lin Qiao's grief directly
